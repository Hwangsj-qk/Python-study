## 모듈(Module)
- 변수와 함수들의 모음을 의미함
  - 어떤 패키지는 함수가 너무 많아서 비슷한 함수끼리 묶어서 여러 개의 모듈로 분리함. 
  - 모듈은 `.py`로 끝나는 파일임
- 모듈 함수 사용법
  1. 모듈명.함수명() 사용
  2. 함수명()으로 사용 → 단일 함수를 쓸 때
  3. 별명 지정하여 사용
```python
# sklearn의 metrics 모듈 로드
import sklearn.metrics

## 1. 모듈명.함수명() 사용
sklearn.metrics.accuracy_score()

## 2. 함수명()으로 사용 → 단일 함수 사용시
from sklearn.metrics import accuracy_score, precision_score
accuracy_score()
precision_score()

## 3. 별명을 지정하여 사용
import sklearn.metrics as met
met.accuracy_score()
```

## 데이터 프레임(Data Frame)
- 데이터를 다룰 때 가장 많이 사용하는 데이터 형식
- 행과 열로 구성된 사각형 모양의 표처럼 구성된 데이터
  - '열'은 속성을 의미 (= 컬럼, 변수)
  - '행'은 전체 정보를 의미 (= 로우, 케이스)
  - 한 사람의 정보는 가로 한 줄에 나열
  - 데이터가 크다 = 행 또는 열이 많다. → 행보다 열이 많은 것이 중요
    - 행이 많다 → 컴퓨터가 느려짐 → 고사양 장비 구축으로 해결 가능
    - 열이 많다 → 분석 방법의 한계 발생 → 고급 분석 방법 모색
- 데이터 프레임 생성 : `pandas` 패키지 사용

## 데이터 정제 
- 데이터 오류를 찾아내는 과정
1. 결측치 : 누락된 값, 비어있는 값을 의미 
   - 파이썬에서 결측치는 **NaN**으로 표시
   - 결측치는 하나라도 있으면 제거하는 것이 맞지만, 데이터 손실이 클 땐 사용 X
   - 결측치 대체 → **평균값**으로 대체
2. 이상치 : 이상한 수치, 정산범위에서 크게 벗어난 값
   - 이상치 제거 : 존재할 수 없는 값 제거 (ex. 사람 키가 3m가 넘는 경우, 나이가 200살이 넘는 경우 등)
   - 이상치 제거 방법
   1. 이상치 확인
   2. 결측 처리
   3. 결측치 제외 분석
