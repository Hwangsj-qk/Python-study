## 변수
- 변수는 변하는 수를 의미하며 파이썬에서는 ***=*** 등호를 활용
- 변수는 데이터 분석의 대상임
- 반면 상수의 경우 하나의 값으로만 되어 있는 속성

### 변수명 규칙
1. 변수명은 알아보기 쉽고 잘 기억할 수 있게 의미를 담아서 지정
2. 언더바(_) 사용하여 변수명을 지을 수 있다 → `스네이크 케이스 법칙`
3. 반드시 문자로 지정해야 한다.
4. 변수명은 한글로 정해도 되지만, 간호 오류가 날 때가 있으므로 ***영문을 권장함***
5. 변수는 대소문자를 구분함 → 소문자로 짓는 습관이 좋다.

## 문자로된 변수 생성

  - 문자열: 문장, 단어를 넣을 수 있고, 띄어쓰기나 특수문자 또한 인정됨

### 리스트

- 문자열일 경우, 대괄호 안에 따옴표 혹은 쌍따옴표를 이용하여 값 입력

## 패키지(Package)

- 패키지 사용 방법 : 패키지 설치 → 패키지 로드 → 패키지 내 함수 사용

## 모듈(Module)
- 변수와 함수들의 모음을 의미함
  - 어떤 패키지는 함수가 너무 많아서 비슷한 함수끼리 묶어서 여러 개의 모듈로 분리함. 
  - 모듈은 `.py`로 끝나는 파일임
- 모듈 함수 사용법
  1. 모듈명.함수명() 사용
  2. 함수명()으로 사용 → 단일 함수를 쓸 때
  3. 별명 지정하여 사용
```python
# sklearn의 metrics 모듈 로드
import sklearn.metrics

## 1. 모듈명.함수명() 사용
sklearn.metrics.accuracy_score()

## 2. 함수명()으로 사용 → 단일 함수 사용시
from sklearn.metrics import accuracy_score, precision_score
accuracy_score()
precision_score()

## 3. 별명을 지정하여 사용
import sklearn.metrics as met
met.accuracy_score()
```

## 데이터 프레임(Data Frame)
- 데이터를 다룰 때 가장 많이 사용하는 데이터 형식
- 행과 열로 구성된 사각형 모양의 표처럼 구성된 데이터
  - '열'은 속성을 의미 (= 컬럼, 변수)
  - '행'은 전체 정보를 의미 (= 로우, 케이스)
  - 한 사람의 정보는 가로 한 줄에 나열
  - 데이터가 크다 = 행 또는 열이 많다. → 행보다 열이 많은 것이 중요
    - 행이 많다 → 컴퓨터가 느려짐 → 고사양 장비 구축으로 해결 가능
    - 열이 많다 → 분석 방법의 한계 발생 → 고급 분석 방법 모색
- 데이터 프레임 생성 : `pandas` 패키지 사용

## 데이터 파악
- 데이터 파악은 데이터를 다루기 쉽도록 데이터를 수정하는 것

### 데이터 파악 함수
- `head()` : 데이터 윗 부분 출력d
- `tail()` : 데이터 아랫 부분 출력
- `shape` : 행과 열의 갯수 출력
- `info()` : 변수 속성 출력
- `describe()` : 요약 통계량 출력
  - std : 표준편차 → 변수의 값들이 평균에서 떨어진 정도를 나타내는 값
  - 25% : 1사분위수 → 하위 25% 지점에 위치한 값
  - 75% : 3사분위수 → 하우 75% 지점에 위치한 값
